## はじめに

**The Truth**は、ニュース記事の検証と透明性を提供するプラットフォームです。信頼できる情報源の追跡、検証レベルの表示、透明性指標の提供を通じて、ユーザーが正確な情報にアクセスできるよう支援します。

---

## プロジェクト概要

**The Truth**は、以下の特徴を持つニュース検証プラットフォームです：

- **ニュース記事の検証**: 記事の信頼性を多角的に評価
- **透明性の提供**: 情報源、編集履歴、AI使用状況などを公開
- **バイアス分析**: 記事の政治的傾向やイデオロギーを可視化
- **C2PA認証**: メディアの真正性を証明するバッジ表示
- **キーワードタイムライン**: 特定のトピックの時系列追跡
- **多言語対応**: 日本語と英語に対応

## システム要件

### 必要なソフトウェア

- **Node.js**: バージョン 18.0.0 以上
- **npm** または **bun**: パッケージマネージャー
- **モダンブラウザ**: Chrome、Firefox、Safari、Edge の最新版

### 推奨環境

- **OS**: Windows 10/11、macOS 10.15以上、Linux（Ubuntu 20.04以上）
- **メモリ**: 4GB以上
- **ディスク容量**: 500MB以上の空き容量

## インストール方法

### 1. リポジトリのクローン

```bash
git clone https://github.com/Savrust/truthseeker-lab.git
cd truthseeker-lab
```

### 2. 依存関係のインストール

```bash
npm install
```

または

```bash
bun install
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスすると、アプリケーションが表示されます。

### 4. 本番用ビルド

```bash
npm run build
```

ビルドされたファイルは `dist` ディレクトリに生成されます。

## 基本的な使い方

1. アプリケーションを起動すると、イントロダクションビデオが表示されます
2. ビデオを視聴するか、「Skip」ボタンでスキップできます

- **フッター**: サイト情報、リンク、連絡先情報

## 主要機能

### 1. 記事の発見と閲覧

#### Discover（発見）タブ

- **記事一覧**: 最新の検証済みニュース記事を表示
- **検索機能**: キーワードで記事を検索
- **フィルタリング**: カテゴリ、検証レベル、日付で絞り込み
- **記事カード**: 各記事の概要、検証レベル、ソース情報を表示

#### 記事の詳細表示

記事カードをクリックすると、詳細情報が表示されます：

- **記事本文**: 完全な記事内容
- **検証レベル**: 完全検証、部分検証、未検証の表示
- **C2PAバッジ**: メディアの真正性証明
- **ソース情報**: 一次ソース、二次ソースの表示
- **バイアス分析**: リベラル、中立、保守の割合
- **イデオロギーバッジ**: 記事の政治的傾向
- **編集履歴**: 記事の作成・更新履歴
- **修正履歴**: 記事の訂正情報
- **AI使用状況**: AIが使用された箇所と割合
- **脚注**: 情報源の詳細な引用

### 2. 記事の検証

#### Verify（検証）タブ

- **検証キュー**: 検証待ちの記事を管理
- **検証ケース**: 個別の検証案件を追跡
- **証拠の管理**: 支持証拠と反証拠の整理
- **タイムライン再構築**: 事件の時系列を追跡
- **協調拡散マップ**: 同時投稿や画像類似性の検出

### 3. キーワードタイムライン

- **キーワード追加**: 追跡したいトピックを追加
- **時系列表示**: キーワードに関連する出来事を時系列で表示
- **イベント追跡**: 発生→反応→訂正→反証/確認の流れを追跡

### 4. 検索機能

- **キーワード検索**: 記事タイトル、本文、タグから検索
- **高度な検索**: 複数の条件を組み合わせて検索
- **検索結果のフィルタリング**: 検証レベル、日付範囲などで絞り込み

### 5. お気に入り機能

- **記事の保存**: 興味のある記事をお気に入りに追加
- **お気に入り一覧**: 保存した記事を一覧表示
- **キーワードの保存**: 追跡したいキーワードを保存

### 6. チャットボット

- **AIアシスタント**: 画面右下のチャットボタンからアクセス
- **サポート**: サービス、機能、使い方に関する質問に対応
- **リアルタイム応答**: 即座に回答を提供

### 7. アカウント機能

#### ログイン

- **メールアドレスとパスワード**: 既存アカウントでログイン
- **ログイン状態の保持**: セッション管理

#### 新規登録

- **アカウント作成**: メールアドレス、パスワード、ユーザー名で登録
- **メール確認**: 登録確認メールの送信（実装により異なる）

#### サブスクリプション

- **プラン選択**: 利用可能なサブスクリプションプランを表示
- **支払い**: PayPalなどの決済方法で支払い
- **機能制限**: 無料ユーザーは一部機能が制限されます

## ページ説明

### ホームページ（/）

- **ヒーローバナー**: プラットフォームの紹介
- **記事一覧**: 注目記事の表示
- **タブナビゲーション**: Discover/Verifyの切り替え
- **お客様の声**: ユーザーテストimonials

### ログインページ（/login）

- **ログインフォーム**: メールアドレスとパスワードでログイン
- **新規登録リンク**: アカウント作成ページへのリンク
- **パスワードリセット**: パスワードを忘れた場合のリンク

### 登録ページ（/register）

- **登録フォーム**: 新規アカウントの作成
- **利用規約**: 利用規約への同意
- **プライバシーポリシー**: プライバシーポリシーの確認

### サブスクリプションページ（/subscription）

- **プラン一覧**: 利用可能なサブスクリプションプランの表示
- **機能比較**: 各プランの機能比較
- **支払い方法**: 利用可能な決済方法

### 支払いページ（/payment）

- **支払いフォーム**: クレジットカード情報の入力
- **PayPal統合**: PayPalでの支払いオプション
- **支払い確認**: 支払い処理の確認

### 検索ページ（/search）

- **検索バー**: キーワード入力
- **検索結果**: 検索結果の一覧表示
- **フィルター**: 検証レベル、日付、カテゴリで絞り込み

### Truthページ（/truth）

- **分類軸**: 記事を複数の軸で分類
  - 基本: 欧米/非欧米
  - リージョン: アジア、欧州、オセアニア、中東、アフリカ、北米/中南米
  - 粒度: 精密/粗い
  - 表示タイプ: 多数派/少数派
- **ソース一覧**: 主要メディアソースの表示
- **記事マトリックス**: 分類に基づいた記事の配置

### 404ページ（/*）

- **エラーメッセージ**: ページが見つからない場合の表示
- **ホームへのリンク**: ホームページに戻るリンク

## よくある質問

### Q1: 無料で利用できますか？

A: はい、基本的な機能は無料で利用できます。ただし、詳細な記事内容や検証情報の一部は、サブスクリプションが必要な場合があります。

### Q2: 検証レベルとは何ですか？

A: 記事の信頼性を示す指標です：
- **完全検証**: 十分な証拠があり、信頼性が高い
- **部分検証**: 一部の情報が検証済み
- **未検証**: まだ検証が完了していない

### Q3: C2PAバッジとは何ですか？

A: メディアの真正性を証明する認証バッジです。画像や動画が改ざんされていないことを示します。

### Q4: 記事のバイアスはどのように判定されますか？

A: 記事の内容、情報源、表現を分析し、リベラル、中立、保守の割合を算出します。

### Q5: キーワードタイムラインはどのように使いますか？

A: 「Discover」タブで「Add Keyword」ボタンをクリックし、追跡したいキーワードを追加します。その後、そのキーワードに関連する出来事が時系列で表示されます。

### Q6: チャットボットは24時間利用できますか？

A: はい、チャットボットは24時間利用可能です。画面右下のチャットアイコンをクリックしてアクセスできます。

### Q7: 言語を変更するには？

A: ヘッダーの言語切り替えボタンから、日本語と英語を切り替えることができます。

### Q8: 記事を共有するには？

A: 記事詳細ページの「Share」ボタンから、ソーシャルメディアやメールで共有できます。

## トラブルシューティング

### 問題: アプリケーションが起動しない

**解決方法:**
1. Node.jsが正しくインストールされているか確認
2. `npm install`を再実行
3. ポート5173が使用されていないか確認
4. ブラウザのキャッシュをクリア

### 問題: 記事が表示されない

**解決方法:**
1. インターネット接続を確認
2. ブラウザのコンソールでエラーを確認
3. ページをリロード
4. ログイン状態を確認（一部の記事はログインが必要）

### 問題: ログインできない

**解決方法:**
1. メールアドレスとパスワードが正しいか確認
2. 大文字・小文字を確認
3. アカウントが有効か確認
4. パスワードリセットを試す

### 問題: チャットボットが表示されない

**解決方法:**
1. ブラウザのJavaScriptが有効か確認
2. ページをリロード
3. ブラウザの拡張機能を無効化して試す
4. 別のブラウザで試す

### 問題: 検索結果が表示されない

**解決方法:**
1. 検索キーワードを確認
2. スペルミスがないか確認
3. より一般的なキーワードで検索
4. フィルターをリセット

### 問題: サブスクリプションの支払いが完了しない

**解決方法:**
1. 決済情報が正しいか確認
2. カードの有効期限を確認
3. 十分な残高があるか確認
4. 別の決済方法を試す
5. サポートに連絡

### 問題: ページの読み込みが遅い

**解決方法:**
1. インターネット接続速度を確認
2. ブラウザのキャッシュをクリア
3. 不要なブラウザタブを閉じる
4. ブラウザを最新版に更新
